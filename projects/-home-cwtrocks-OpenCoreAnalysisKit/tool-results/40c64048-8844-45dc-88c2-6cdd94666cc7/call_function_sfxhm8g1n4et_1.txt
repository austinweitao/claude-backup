   107→# How To Launch
   108→```
   109→Usage: core-parser [OPTION]
   110→Option:
   111→    -c, --core <COREFILE>    load core-parser from corefile
   112→    -p, --pid <PID>          load core-parser from target process
   113→    -m, --machine <ARCH>     arch support arm64, arm, x86_64, x86, riscv64
   114→        --sdk <SDK>          sdk support 26 ~ 36
   115→        --no-filter-any      load core-parser no filter any vma
   116→    -t, --tomb <TOMBSTONE>   load core-parser form tombstone file
   117→        --sysroot <DIR:DIR>  set sysroot path
   118→        --va_bits <BITS>     set virtual valid addr bits
   119→        --page_size <SIZE>   set target core page size
   120→        --no-load            no auto load corefile
   121→        --no-fake-phdr [EXE] rebuild fakecore phdr
   122→Exp:
   123→    core-parser -c /tmp/tmp.core
   124→    core-parser -p 1 -m arm64
   125→    core-parser -t tombstone_00 --sysroot symbols
   126→```
