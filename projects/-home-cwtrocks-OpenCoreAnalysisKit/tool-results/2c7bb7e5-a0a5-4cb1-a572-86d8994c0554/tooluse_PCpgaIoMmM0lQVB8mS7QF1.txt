   121→# How To Launch
   122→```
   123→Usage: core-parser [OPTION]
   124→Option:
   125→    -c, --core <COREFILE>    load core-parser from corefile
   126→    -p, --pid <PID>          load core-parser from target process
   127→    -m, --machine <ARCH>     arch support arm64, arm, x86_64, x86, riscv64
   128→        --sdk <SDK>          sdk support 26 ~ 36
   129→        --no-filter-any      load core-parser no filter any vma
   130→    -t, --tomb <TOMBSTONE>   load core-parser form tombstone file
   131→        --sysroot <DIR:DIR>  set sysroot path
   132→        --va_bits <BITS>     set virtual valid addr bits
   133→        --page_size <SIZE>   set target core page size
   134→        --no-load            no auto load corefile
   135→        --no-fake-phdr [EXE] rebuild fakecore phdr
   136→Exp:
   137→    core-parser -c /tmp/tmp.core
   138→    core-parser -p 1 -m arm64
   139→    core-parser -t tombstone_00 --sysroot symbols
   140→```
